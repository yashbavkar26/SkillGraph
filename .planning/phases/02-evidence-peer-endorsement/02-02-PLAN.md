  <action>Create POST /api/endorse and GET /api/endorse/:userId endpoints.

- POST: CREATE (u1:User {id: $endorserId})-[:ENDORSED {timestamp: $timestamp, comment: $comment}]->(u2:User {id: $recipientId}) and link to Skill.

Validate input using the Zod schema from Task 1.</action>
